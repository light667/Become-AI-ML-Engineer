# Become AI/ML Engineer 🚀

**Roadmap interactive et complète pour devenir AI/ML Engineer Senior — de zéro à un portfolio prêt pour l'embauche.**

Ce dépôt contient une application web interactive (React, single-file, sans build) qui guide pas à pas un développeur Python vers le poste d'AI/ML Engineer Senior, à travers un plan structuré de **9 mois**.

---

## 📖 Aperçu

Cette roadmap transforme un parcours d'apprentissage classique en un tableau de bord visuel et actionnable : progression suivie, checklists journalières, projets phares par étape, salaires indicatifs et ressources recommandées.

| Indicateur | Valeur |
|---|---|
| Durée totale | 9 mois |
| Semaines | 36 |
| Jours de contenu | 252 |
| Étapes de carrière | 5 |
| Mini-projets | 36 (1 par semaine) |
| Projets phares | 5 (1 par étape) |

---

## 🎯 Les 5 étapes de carrière

| # | Étape | Poste visé | Durée | Objectifs clés |
|---|---|---|---|---|
| 1 | 🐍 Étape 1 | Python Developer | 1 mois | Python avancé, tests, APIs, bases de données, architecture logicielle |
| 2 | 🧠 Étape 2 | AI Engineer Junior | 2 mois | Maths pour le ML, Scikit-learn, PyTorch, Transformers, LLMs & RAG |
| 3 | ⚙️ Étape 3 | Senior AI Engineer | 3 mois | LLMOps, architecture IA, sécurité, scalabilité, leadership technique |
| 4 | 📊 Étape 4 | ML Engineer | 2 mois | Data pipelines, feature stores, model serving, A/B testing, ML temps réel |
| 5 | 🏆 Étape 5 | AI/ML Engineer | 1 mois | Spécialisation, projet vedette, leadership, recherche d'emploi senior |

### Projets phares par étape

1. **API de Gestion Professionnelle** — Backend Python complet (FastAPI, SQLAlchemy, PostgreSQL, Docker, CI/CD)
2. **Assistant IA Multi-Documents** — Application RAG complète
3. **Plateforme IA Multi-Agents** — Sécurisée et scalable, prête pour la production
4. **Plateforme MLOps de Recommandation** — Temps réel, à l'échelle
5. **Plateforme AI/ML de Production de Bout en Bout** — Projet vedette final

Chaque semaine se décompose en 7 jours de contenu (focus technique + outils associés) et se conclut par un **mini-projet** pratique qui consolide les compétences acquises.

## ✨ Fonctionnalités de l'application

- **Suivi de progression** : cases à cocher journalières avec calcul automatique du pourcentage d'avancement global
- **Navigation par onglets** : roadmap détaillée, ressources, organisation, opportunités d'emploi
- **Vue par étape et par semaine** : contenu déroulant avec focus quotidien et outils
- **Fiches projets phares** : livrables attendus, stack technique, durée estimée
- **Ressources clés** : sélection organisée par le contenu du programme
- **Marché de l'emploi** : progression des postes avec salaires indicatifs (remote, international, USD) et niveau de demande
- **Checklist portfolio final** et conseils pour réussir le parcours

## 🚀 Utilisation

Aucune installation requise : l'application est un fichier HTML autonome.

```bash
# Cloner le dépôt
git clone https://github.com/<votre-utilisateur>/Become-AI-ML-Engineer.git
cd Become-AI-ML-Engineer

# Ouvrir directement dans le navigateur
open index.html   # macOS
xdg-open index.html   # Linux
start index.html   # Windows
```

Ou lancez un petit serveur local si votre navigateur bloque les scripts en `file://` :

```bash
python -m http.server 8000
# puis ouvrez http://localhost:8000
```
